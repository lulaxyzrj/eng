# Orientações para Execução

### Ambiente de Desenvolvimento:

* IDE de sua preferência
* Cliente ou plugin Git
* [NodeJS v10.17.0](https://nodejs.org/pt/blog/release/v10.17.0).
* [NPM 6.11.3](https://nodejs.org/pt/blog/release/v6.11.3)

Instruções para Execução:

* Em desenvolvimento, a aplicação pode ser executada diretamente na IDE ou pelo comando "npm run start" no terminal, obedecendo as dependências do Node e NPM
